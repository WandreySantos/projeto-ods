# Requisitos do Sistema

## ✅ Requisitos Funcionais (RF)

- RF01: O sistema deve permitir cadastro de usuários
- RF02: O sistema deve permitir login de usuários
- RF03: O usuário deve poder registrar uma denúncia
- RF04: O sistema deve listar denúncias cadastradas
- RF05: O usuário deve poder editar denúncias
- RF06: O usuário deve poder excluir denúncias

---

## ⚙️ Requisitos Não Funcionais (RNF)

- RNF01: O sistema deve ser responsivo
- RNF02: O sistema deve possuir autenticação segura (JWT)
- RNF03: O sistema deve ter tempo de resposta inferior a 2 segundos
- RNF04: O sistema deve seguir arquitetura REST

---

## 👤 Histórias de Usuário

- Como usuário, quero me cadastrar para acessar o sistema
- Como usuário, quero fazer login para acessar minhas funcionalidades
- Como usuário, quero registrar uma denúncia para reportar um problema
- Como usuário, quero visualizar denúncias para acompanhar situações
- Como usuário, quero editar minhas denúncias
- Como usuário, quero excluir denúncias que não são mais relevantes
