## 🏗️ Arquitetura do Sistema

O sistema CityFix será desenvolvido utilizando uma arquitetura cliente-servidor.

## 🔧 Tecnologias Utilizadas

- Frontend: React com TypeScript
- Backend: Spring Boot (Java) com API REST
- Banco de Dados: MySQL

## 📐 Estrutura

- Frontend: Interface do usuário responsável pela interação com o sistema
- Backend: API responsável pelas regras de negócio e comunicação com o banco de dados
- Banco de Dados: Armazenamento persistente das informações

## 💡 Justificativa

O uso de React com TypeScript permite maior organização e segurança no desenvolvimento do frontend.

O Spring Boot foi escolhido por sua robustez, escalabilidade e ampla utilização no mercado para construção de APIs REST.

O MySQL foi selecionado por ser um banco de dados relacional confiável, amplamente utilizado e de fácil integração com o Spring Boot.
