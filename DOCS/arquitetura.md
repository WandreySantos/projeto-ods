# Arquitetura do Sistema

O sistema CityFix será desenvolvido utilizando uma arquitetura cliente-servidor.

## Estrutura

- Frontend: Interface do usuário (React)
- Backend: API REST (Node.js + Express)
- Banco de Dados: MongoDB

## Justificativa

A arquitetura REST foi escolhida por sua simplicidade, escalabilidade e compatibilidade com aplicações web modernas.
