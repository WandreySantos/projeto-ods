# CityFix

## 🌍 ODS Escolhido
ODS 11 – Cidades e Comunidades Sustentáveis

## ❗ Problema
A população enfrenta dificuldades para reportar problemas urbanos como buracos, lixo acumulado e iluminação pública defeituosa, devido à falta de canais eficientes e centralizados.

## 💡 Solução
O CityFix é um sistema web que permite aos usuários registrar, visualizar e acompanhar denúncias de problemas urbanos em sua cidade.

## 🏗️ Tipo de Sistema
Sistema web com arquitetura frontend + backend (API RESTful)

## ⚙️ Tecnologias (previsão)
- Frontend: React
- Backend: Node.js + Express
- Banco de Dados: MongoDB
- Autenticação: JWT

## 📅 Planejamento
No TP2 será definido o modelo arquitetural (C4), tecnologias finais e estrutura do sistema.
